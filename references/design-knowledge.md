## 1. LIFT System

The LIFT framework forces strategic thinking before any visual decision.

### L — Leverage-Asset
The single strongest visual element to build the entire system around. Every other design decision amplifies this.

Examples:
- A memorable letterform (the Volkswagen V, the Google G)
- A symbolic shape with cultural meaning (Nike swoosh = motion/wind)
- A color so ownable it becomes the brand (Tiffany blue, UPS brown)
- A typographic style so distinctive it IS the brand (Supreme, Supreme)

**How to identify it:** Ask "what one thing would people recognize if all other context was stripped away?" If nothing stands out → the brand needs to create a leverage-asset from scratch.

**Common mistakes:**
- Trying to make every element the leverage-asset (nothing is memorable)
- Choosing the most obvious element (letter = first letter of name — boring)
- Confusing "what we do" with "what makes us visually distinctive"

### I — Internal Rhythm
The visual cadence that runs consistently through all brand assets. Creates cohesion without repetition.

Rhythm types:
- **Geometric repeat:** A specific angle (45°), a rounded corner radius, a specific stroke weight used everywhere
- **Organic flow:** A specific curve style, an ink-brush quality, a handmade irregularity
- **Grid cadence:** A modular system where everything snaps to the same underlying grid unit
- **Typographic rhythm:** A specific relationship between headline size and body — 3:1, never 2:1

**How to define it:** Pick ONE rule. "Every graphic element uses 8px radius corners." "All lines are 2px at the same angle." "All compositions leave the right 30% empty."

### F — Friction Sources
What will make this brand feel stuck, generic, or wrong. Identify these before designing to avoid them.

Common friction sources:
- **Category clichés:** Tech brands that use circuit-board patterns, health brands that use leaves, finance brands that use upward arrows
- **AI tells:** Gradient orbs, 3D glassmorphism, neon on dark backgrounds, "geometric abstract" logos that look like every other startup
- **Color mismatch:** Warm, earthy colors for a clinical or precision brand
- **Type mismatch:** Rounded, friendly fonts for a brand whose verbs are Protect, Secure, Enforce
- **Complexity that doesn't scale:** A detailed illustrative logo that becomes noise at 32px

### T — Transferability
How well does the brand identity survive across all use cases?

Test questions:
- **Small:** Does the logo work as a 32×32 favicon?
- **Large:** Does it work at 3 meters wide on a banner?
- **Dark background:** Does it work inverted? (Need a white/light variant ready?)
- **1-color:** Does it work in single-color for embroidery, stamps, embossing?
- **Motion:** Could it animate logically? (A mark that has movement implied in it)
- **International:** Does the symbol or name have unintended meanings in other languages/cultures?

## 2. 12 Grid Archetypes

Choose ONE grid for the brand. All layouts derive from it.

### 1. Manuscript Grid
```
┌─────────────────────────┐
│  ░░░░░░░░░░░░░░░░░░░░  │
│  ░░░░░░░░░░░░░░░░░░░░  │
│  ░░░░░░░░░░░░░░░░░░░░  │
│  ░░░░░░░░░░░░░░░░░░░░  │
└─────────────────────────┘
```
Single content column. Used for: books, editorials, long-form brands, law firms.

### 2. Column Grid (even)
```
┌───┬───┬───┬───┬───┬───┐
│ ░ │ ░ │ ░ │ ░ │ ░ │ ░ │
└───┴───┴───┴───┴───┴───┘
```
6 or 12 equal columns. Used for: newspapers, grids with many content types.

### 3. Modular Grid
```
┌───┬───┬───┐
│ ░ │ ░ │ ░ │
├───┼───┼───┤
│ ░ │ ░ │ ░ │
├───┼───┼───┤
│ ░ │ ░ │ ░ │
└───┴───┴───┘
```
Equal rows AND columns. Used for: data dashboards, catalogs, Swiss design.

### 4. Hierarchical Grid
```
┌─────────────────────────┐
│       HERO (wide)       │
├───────────┬─────────────┤
│  content  │   sidebar   │
└───────────┴─────────────┘
```
Columns of unequal width based on content hierarchy. Used for: magazines, editorial brands with strong visual hierarchy.

### 5. Baseline Grid
Horizontal rhythm defined by line height unit (8px, 12px, etc.). All elements snap to it. Used for: typography-led brands, bookish aesthetics.

### 6. Axial Grid
Everything aligns to a single vertical or horizontal axis. Strong asymmetric energy. Used for: fashion brands, architectural firms, luxury.

### 7. Radial Grid
Elements arranged around a central point. Used for: circular logos, mandala-inspired brands, food brands.

### 8. Disordered Grid (Intentional Chaos)
Random placement with controlled size relationships. Used for: youth brands, counterculture, creative agencies, music.

### 9. Split Grid (50/50)
Two equal halves, often different colors or image vs. type. Used for: before/after, contrast brands, dual-market products.

### 10. Bleeding Grid
Images and elements bleed to the edge with no safe margins. Maximum energy. Used for: sports brands, entertainment, action products.

### 11. Swiss/International Grid
Strict mathematical grid, maximum white space, objective. Used for: tech, pharmaceutical, precision engineering.

### 12. Pattern Grid
A repeating motif fills the ground — logos, shapes, or textures. Used for: packaging, luxury goods, streetwear.

## 3. Verb-Based Design

The verb-trio is the most important strategic input. Every visual decision traces back to it.

### Core Verb Categories

**PROTECT** family: Secure, Guard, Shield, Defend, Preserve, Fortify
→ Visual: strong geometry, enclosed shapes, thick strokes, dark anchor colors, stability cues (horizontal composition)

**CONNECT** family: Link, Bridge, Network, Unite, Weave, Bond
→ Visual: continuous lines, interlocking forms, circular compositions, warm colors, organic curves

**ACCELERATE** family: Drive, Launch, Thrust, Propel, Amplify, Surge
→ Visual: diagonal motion cues, sharp angles, speed lines, bold accent color, asymmetric tension

**NURTURE** family: Grow, Cultivate, Sustain, Heal, Support, Flourish
→ Visual: organic shapes, earthy tones, soft edges, upward-reaching forms, space and breathing room

**DISRUPT** family: Break, Challenge, Rewrite, Subvert, Transform, Reinvent
→ Visual: broken grids, unexpected color combinations, mixed type scales, collage energy, rule-breaking composition

**CLARIFY** family: Reveal, Illuminate, Simplify, Decode, Focus, Cut-Through
→ Visual: high contrast, minimal palette, strong negative space, clean sans-serif type, beam/spotlight metaphors

**CREATE** family: Build, Craft, Make, Invent, Design, Forge
→ Visual: hand-crafted textures, tool/process metaphors, raw materials visual language, workshop aesthetic

**ELEVATE** family: Premium, Luxury, Refine, Ascend, Distinguish, Excel
→ Visual: restrained palette, generous white space, thin-weight typography, gold/silver accents, minimal ornamentation

### Verb Tension (the most interesting brands have verbs in tension)
- Protect + Accelerate → Security that moves fast (→ sharp + stable hybrid)
- Nurture + Disrupt → Compassionate disruption (→ organic shapes with bold color)
- Clarify + Create → Making complexity beautiful (→ Swiss grid + craft texture)

## 4. Typography Pairing Rules

**Rule 1: Maximum contrast, minimum confusion**
Pair fonts that are clearly different in category (Serif + Sans, Display + Text) but share a similar x-height or proportional feel.

**Rule 2: One font leads, one supports**
Never pair two display fonts. One font is the "voice" (display/headlines), one is the "body" (reading/text).

**Reliable pairings for brand booklets:**

| Display | Body | Vibe |
|---|---|---|
| Clash Display | Satoshi | Modern, geometric, confident |
| Cabinet Grotesk | Plus Jakarta Sans | Premium editorial |
| Syne | DM Sans | Creative studio |
| Space Grotesk | Inter | Tech, neutral |
| Playfair Display | Source Sans | Editorial luxury |
| Anton | Roboto | Bold, sports, impact |
| Bebas Neue | Open Sans | Poster, streetwear |
| Cormorant | Jost | Refined fashion |

**Rule 3: Stick to 2 fonts max for brand systems**
More = chaos. The brand's voice is in the display font; legibility is in the body font.

**Rule 4: Size ratio ≥ 1.25**
If body is 16px, headline should be at minimum 20px. Flat scale = dead hierarchy.

## 5. Color Theory for Brands

**The 60/30/10 Rule:**
- 60% = dominant neutral (background, large surfaces)
- 30% = secondary (main brand color, most UI elements)
- 10% = accent (calls-to-action, highlights, pops)

**OKLCH vs. HEX:**
Modern brand systems use OKLCH for perceptually uniform color. When specifying colors, provide both HEX (for legacy tools) and OKLCH (for design systems).

**Contrast minimums:**
- Body text on background: 4.5:1 minimum (WCAG AA)
- Large text/logos: 3:1 minimum
- Decorative elements: No requirement (but don't strain eyes)

**Color personality:**
| Color family | Psychological signal | Brand categories |
|---|---|---|
| Deep Blue | Trust, stability, expertise | Finance, law, tech |
| Forest Green | Growth, sustainability, health | Wellness, food, eco |
| Warm Red/Orange | Energy, urgency, appetite | Food, sports, retail |
| Purple | Creativity, luxury, mystery | Fashion, beauty, entertainment |
| Black/Charcoal | Sophistication, authority, premium | Luxury, fashion, tech |
| Gold/Amber | Premium, warmth, craftsmanship | Luxury goods, artisan |

**Avoid:**
- More than 3 named brand colors (creates confusion)
- Saturated colors at full saturation (looks cheap/digital)
- Neon colors unless the brand explicitly owns countercultural positioning

## 6. Modernist vs. Maximalist Decision Tree

```
Does the brand need to communicate:
├── Precision, reliability, expertise, trust?
│   └── → MODERNIST
│       ├── Swiss/International grid
│       ├── Tight type hierarchy
│       ├── 1-2 brand colors max
│       └── Ample negative space
│
└── Energy, personality, abundance, celebration?
    └── → MAXIMALIST
        ├── Disordered or pattern grid
        ├── Multiple type sizes in tension
        ├── 3-4 colors + pattern + texture
        └── Full bleed, no safe margins
```

**Most brands should be 70% Modernist, 30% Maximalist** unless the category demands otherwise (streetwear = 90% max, pharmaceutical = 99% modern).

## 7. AI Anti-Patterns

These are the most common ways AI-generated brand assets fail. Avoid all of them.

### Visual Anti-Patterns
- **Gradient Soup:** Random color gradients applied to everything. It's the default AI aesthetic because gradients are easy to generate. Always specify "no gradients" explicitly.
- **The Orb:** A glowing 3D sphere as the logo symbol. Unless your brand is literally a planet, never.
- **Generic Globe:** Wireframe globe = "we are global." Banned.
- **Circuit Board:** Used for any tech brand. Completely overused. Banned.
- **Leaf/Plant:** Used for any health/eco brand. Banned unless you are literally a plant company.
- **Upward Arrow:** Used for any finance or growth brand. Banned.
- **Lightning Bolt:** Used for any energy or speed brand. Banned unless subverted.
- **Hexagonal Honeycomb Pattern:** Overused in tech. Banned.

### Logo Anti-Patterns
- **Dropping the initials in a circle:** Too generic. At minimum add a meaningful relationship between the letters.
- **Wordmark with no visual distinction:** Plain text in a stock font = not a logo.
- **Symbol that requires explanation:** If you have to explain what the symbol means, it's failing.
- **Competing elements:** Logo with symbol + icon + badge + text = visual noise.

### Color Anti-Patterns
- **The AI Purple-Blue:** `#6B3FA0`-to-`#2563EB` gradient on dark background. Seen on 40% of AI startup logos.
- **Neon on black:** Cyberpunk default. Banned unless explicitly required by brand.
- **Pure black/white:** Always add a slight warm or cool tint.

## 8. Logo System Anatomy

A complete logo system has 4 variants minimum:

### Primary Mark (most important)
- The canonical version
- Used on white/light backgrounds
- Has both symbol and wordmark in the brand's optimal relationship

### Horizontal Lockup
- Symbol left, wordmark right
- Used in headers, nav bars, document letterheads
- Aspect ratio approximately 3:1 to 5:1 width:height

### Vertical/Stacked Lockup
- Symbol top, wordmark bottom
- Used in profile images, small square contexts
- Aspect ratio approximately 1:1 to 2:3 width:height

### Monogram/Mark Only
- Symbol or lettermark isolated, no wordmark
- Used at very small sizes, embossing, watermarks, favicons
- Must read as a single cohesive element at 32px

### Optional: Dark Background Variant
- White or light version of primary mark
- Used on colored, dark, or photographic backgrounds
